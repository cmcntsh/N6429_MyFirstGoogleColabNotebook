# My First Google Colab Notebook

You can see a video demonstration of this assignment at https://youtu.be/CqFbZDLBzNw.

## Step 1: Fork this repository

We did this in a previous assignment. You can review how to fork a repository at https://github.com/cmcntsh/N6429_GitHubForClassAssignments#fork-this-repository.

## Step 2: Create a new Google Colab notebook

* Go to https://colab.research.google.com/
* Create a new notebook and call it FirstGoogleColabNotebook

## Step 3: Enter Python code and run it

* Enter the code `print ("Hello world!")` in the first cell and run it.
* Create a new code block below the first. Enter the following code in the new code block:

```
x = 2 + 2
print (x)
```
* Run the code in the second code block.

## Step 4: Save & Download

* Save the notebook: File - Save
* Download the notebook to your computer: File - Download - Download .ipynb

## Step 5: Upload the notebook to your GitHub repository

We saw how to do this in a previous assignment. You can review how to upload files to GitHub at https://github.com/cmcntsh/N6429_GitHubForClassAssignments#upload-files-directly-to-github

## Step 6: Submit the link to your repository in Canvas

Copy the address from your browser url window and submit it in Canvas.
